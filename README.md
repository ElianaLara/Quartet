`# Quartet

Quartet is a web application utilising Gemini AI to ask users questions. The AI picks out simmilar keywords for the algorithm sorts users into groups of four depending on their shared hobbies and traits, suggesting meet up time and location.

## Features
- User registration and login
- Profile creation and editing
- AI to generate keywords from conversations with users
- Weighted algorithm creating groups

## Tech Stack

Backend: Python, Flask

Database: MongoDB Atlas

Frontend: HTML

## Installation

1. Clone the repository 
   2. git clone https://github.com/ElianaLara/Quartet
3. Ask us for Gemini and MongoDB keys
3. Create a virtual environment 
   4. python -m venv venv
5. Activate the virtual environment 
6. Windows (PowerShell)
   7. .\venv\Scripts\Activate.ps1 
8. Windows (cmd)
   9. venv\Scripts\activate

## Install dependencies
- pip install -r requirements.txt

## Run the application

- python run.py


## Environment Setup

You will need:

- Python 3.10+

- MongoDB running locally or remotely
